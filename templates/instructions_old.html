How to use:

EC Analysis - Calculates peak heights from selected EC data

Microplate Analysis - Generates graphs from microplate data

Detailed instructions - EC Analysis

1. Export EC-Labs files as .mpt

1. Go to experiment -> Export as Text

2. Select files

3. Make sure template is "EC-Lab Text Format (*.mpt)*

- This should be the default format

4. Hit export

5. All files will be exported as .mpt files

2. Go to fredsense.pythonanywhere.com and select "EC Analysis"

3. Push "Choose Files" button

4. Select the .mpt files you wish to analyse (generated in step 1)

- Note: There is a 20mb upload limit on the server (~20 files)

5. Push "Compute" button

- May take a few seconds to upload files

6. Data will appear underneath buttons

- Sample Name = File name

- Peak Height = Peak height calculated from cycle 3 of that sample

Detailed instructions - Microplate Analysis

1. Generate blank and test files (raw .txt from spec) and Layout files (excel files describing location of samples)

2. Push "Blank File" button, select blank data file

3. Push "Data File" button, select data file

4. Push "Layout File" button, select layout file

5. Push "Compute" button

- May take a few seconds to upload files

6. Data will appear underneath buttons
